1. Justify-content: Determines how items are spaced along the main axis, like pushing them to the start, end, or center.

2. Align-items: Controls how items align along the cross-axis, whether at the top, bottom, center, or stretched to fill the container.

3. Align-self: Overrides the alignment set by align-items for individual items within the flex container.

4. Flex-direction: Decides if items should be laid out in a row, column, or reversed versions of these.

5. Flex-wrap: Specifies whether items should stay in a single line or wrap onto the next line if there isn't enough space.
